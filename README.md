# Iranian politicians twitter dataset persian
    
دیتاست کامل توییت های سیاسیون ایرانی در توییتر برای کارهای پردازش متن

## Installing

**Git:**
```bash
git clone https://github.com/twintproject/twint.git
cd twint
pip3 install . -r requirements.txt
```

**Pip:**
```bash
pip3 install twint
```
